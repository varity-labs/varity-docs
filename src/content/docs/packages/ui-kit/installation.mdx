Get the Varity UI Kit installed in your React project.

## Install Packages

<Tabs>
  <TabItem label="npm">
    ```bash
    npm install @varity-labs/ui-kit @varity-labs/sdk
    ```
  </TabItem>
  <TabItem label="pnpm">
    ```bash
    pnpm add @varity-labs/ui-kit @varity-labs/sdk
    ```
  </TabItem>
  <TabItem label="yarn">
    ```bash
    yarn add @varity-labs/ui-kit @varity-labs/sdk
    ```
  </TabItem>
</Tabs>

## Peer Dependencies

The UI Kit requires these peer dependencies:

```json
{
  "react": "^18.0.0",
  "react-dom": "^18.0.0"
}
```

Install any missing peer dependencies:

```bash
npm install react react-dom
```

<Aside type="tip">
`@privy-io/react-auth` and `@tanstack/react-query` are included as regular dependencies. You don't need to install them separately.
</Aside>

## Credentials

No credential setup needed. Varity provides authentication and storage credentials automatically.

<Aside type="tip">
Just install the packages and start building. The UI Kit uses managed credentials by default. See [Managed Credentials](/deploy/managed-credentials) for details.
</Aside>

## Basic Setup

<Steps>

1. **Set up your auth provider**

   ```tsx title="app/layout.tsx"
   import { AuthProvider } from '@varity-labs/ui-kit';

   export default function RootLayout({ children }) {
     return (
       <html>
         <body>
           <AuthProvider appId={process.env.NEXT_PUBLIC_VARITY_AUTH_APP_ID}>
             {children}
           </AuthProvider>
         </body>
       </html>
     );
   }
   ```

2. **Add a login button**

   ```tsx title="components/Header.tsx"
   import { LoginButton, useAuth } from '@varity-labs/ui-kit';

   export function Header() {
     const { authenticated, logout } = useAuth();

     return (
       <header>
         {authenticated ? (
           <button onClick={logout}>Sign Out</button>
         ) : (
           <LoginButton />
         )}
       </header>
     );
   }
   ```

3. **Protect routes**

   ```tsx title="app/dashboard/page.tsx"
   import { ProtectedRoute } from '@varity-labs/ui-kit';

   export default function DashboardPage() {
     return (
       <ProtectedRoute>
         <Dashboard />
       </ProtectedRoute>
     );
   }
   ```

</Steps>

## Framework Setup

### Next.js (App Router)

```tsx title="app/layout.tsx"
import { AuthProvider } from '@varity-labs/ui-kit';

export default function RootLayout({
  children,
}: {
  children: React.ReactNode;
}) {
  return (
    <html lang="en">
      <body>
        <AuthProvider appId={process.env.NEXT_PUBLIC_VARITY_AUTH_APP_ID}>
          {children}
        </AuthProvider>
      </body>
    </html>
  );
}
```

### React (Vite)

```tsx title="src/main.tsx"
import React from 'react';
import ReactDOM from 'react-dom/client';
import { AuthProvider } from '@varity-labs/ui-kit';
import App from './App';

ReactDOM.createRoot(document.getElementById('root')!).render(
  <React.StrictMode>
    <AuthProvider appId={import.meta.env.VITE_VARITY_AUTH_APP_ID}>
      <App />
    </AuthProvider>
  </React.StrictMode>
);
```

### Create React App

```tsx title="src/index.tsx"
import React from 'react';
import ReactDOM from 'react-dom/client';
import { AuthProvider } from '@varity-labs/ui-kit';
import App from './App';

const root = ReactDOM.createRoot(
  document.getElementById('root') as HTMLElement
);

root.render(
  <React.StrictMode>
    <AuthProvider appId={process.env.REACT_APP_VARITY_AUTH_APP_ID}>
      <App />
    </AuthProvider>
  </React.StrictMode>
);
```

## Configuration Options

`AuthProvider` accepts these configuration options:

```tsx
<AuthProvider
  // Optional (uses dev defaults if not set)
  appId="your-varity-auth-app-id"

  // Login methods to show
  loginMethods={['email', 'google', 'twitter', 'discord']}

  // Appearance customization
  appearance={{
    theme: 'light', // or 'dark'
    accentColor: '#2563EB',
    logo: '/logo.png',
  }}

  // Callback when account address changes
  onAddressChange={(address) => console.log('Address:', address)}
>
  {children}
</AuthProvider>
```

## TypeScript Support

The UI Kit is fully typed. Import types as needed:

```typescript
import type {

  LoginButtonProps,
  PaymentWidgetProps,
} from '@varity-labs/ui-kit';
```
