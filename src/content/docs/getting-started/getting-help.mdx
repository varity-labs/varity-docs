## Community

Our Discord is the fastest way to get help from the team and other developers.

[Join Discord](https://discord.gg/7vWsdwa2Bg)

Channels: **#help** for questions, **#general** for discussion, **#showcase** to share what you build.

## Report Issues

Found a bug or have a feature request? Reach out on GitHub or Discord.

[Varity on GitHub](https://github.com/varity-labs)

When reporting bugs, include:

- Steps to reproduce
- Expected vs actual behavior
- OS, Node version, and package versions
- Error messages or logs

## Security Issues

**Do not report security vulnerabilities publicly.**

Email **security@varity.so** instead. We typically respond within 24 hours.

## Feedback

We welcome bug reports, documentation improvements, and feature requests. The fastest way to reach us is Discord, or open an issue on [GitHub](https://github.com/varity-labs).

## Response Times

| Channel | Typical Response | Best For |
|---------|-----------------|----------|
| Discord | Typically a couple of hours | Quick questions |
| GitHub Issues | Typically within a day | Bugs and feature requests |
| Email | Typically within 24 hours | General inquiries |
