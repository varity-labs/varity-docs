## Report Issues

### Bug Reports

Found a bug? Report it on GitHub:

[Report a Bug →](https://github.com/varity-os/varity-sdk/issues/new?template=bug_report.md)

**Include in your report:**
- Steps to reproduce the issue
- Expected vs actual behavior
- Environment details (OS, Node version, package versions)
- Error messages or logs

### Feature Requests

Have an idea for a new feature?

[Request a Feature →](https://github.com/varity-os/varity-sdk/issues/new?template=feature_request.md)

Describe:
- The problem you're trying to solve
- Your proposed solution
- Any alternatives you've considered

## Security Issues

**Do NOT report security vulnerabilities publicly.**

For security issues, email: security@varity.dev

We'll respond within 24 hours and work with you to resolve the issue responsibly.
