<Aside type="caution">
The public MVP scope is the deployment toolchain only: `varitykit` (PyPI) + `@varity-labs/mcp` (npm).
App Store listing and purchase flows are not part of the current shipped user path.
</Aside>

During MVP, use these flows:

- Deploy with `varitykit app deploy`
- Or deploy from an AI editor via `@varity-labs/mcp@beta`
- Access deployed apps at `https://varity.app/<name>/`

This page intentionally omits submission, revenue-split, and purchase-flow guidance because those surfaces are not currently shipped for launch.
