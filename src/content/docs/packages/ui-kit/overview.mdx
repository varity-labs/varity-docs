# @varity/ui-kit

A React component library that handles authentication, payments, and account management so you can focus on building your application.

## What It Does

- **Authentication** - Email, Google, Twitter, Discord, GitHub login - no wallet popups
- **Payments** - Buy USDC with credit card, Apple Pay, or Google Pay
- **Account Management** - Automatic account creation and session handling
- **Zero Gas Fees** - Users never pay transaction fees

<Aside type="tip">
Users sign in with email or social accounts. They never need to install browser extensions or understand cryptocurrency.
</Aside>

## Quick Start

```bash
npm install @varity/ui-kit @varity/sdk
```

```tsx
import { PrivyStack, PrivyLoginButton } from '@varity/ui-kit';

function App() {
  return (
    <PrivyStack>
      <PrivyLoginButton />
      <YourApp />
    </PrivyStack>
  );
}
```

That's it. Users can now sign in with email or social accounts.

## How Authentication Works

1. User clicks "Sign In"
2. Modal appears with email/social options
3. User enters email (receives code) or clicks social login
4. Account is created automatically
5. User is authenticated - ready to use your app

```tsx
import { usePrivy } from '@varity/ui-kit';

function Dashboard() {
  const { authenticated, user, logout } = usePrivy();

  if (!authenticated) {
    return <PrivyLoginButton />;
  }

  return (
    <div>
      <p>Welcome, {user?.email?.address || 'User'}</p>
      <button onClick={logout}>Sign Out</button>
    </div>
  );
}
```

## What Gets Created Automatically

When a user signs in with email or social login:

| Behind the Scenes | What User Sees |
|-------------------|----------------|
| Smart wallet created | "Signed in successfully" |
| Wallet linked to account | User profile with email |
| Ready for transactions | Can use your app immediately |

Users never see wallet addresses, private keys, or transaction confirmations unless you choose to show them.

## Package Exports

### Providers

| Export | Description |
|--------|-------------|
| `PrivyStack` | All-in-one provider (recommended) |
| `VarityPrivyProvider` | Privy authentication with Wagmi |
| `VarityDashboardProvider` | Dashboard-specific provider setup |
| `WalletSyncProvider` | Syncs Privy wallets with thirdweb |

| Export | Description |
|--------|-------------|

## Production Example

```tsx
import {
  PrivyStack,
  PrivyLoginButton,
  PrivyUserProfile,
  PrivyProtectedRoute,
  useWalletSync,
} from '@varity/ui-kit';

function App() {
  return (
    <PrivyStack
      appId={process.env.NEXT_PUBLIC_PRIVY_APP_ID}
      thirdwebClientId={process.env.NEXT_PUBLIC_THIRDWEB_CLIENT_ID}
      loginMethods={['email', 'google']}
      appearance={{
        theme: 'light',
        accentColor: '#2563EB',
        logo: '/logo.png',
      }}
    >
      <Layout>
        <Header />
        <PrivyProtectedRoute>
          <Dashboard />
        </PrivyProtectedRoute>
      </Layout>
    </PrivyStack>
  );
}

function Header() {
  const { isAuthenticated, address } = useWalletSync();

  return (
    <header>
      {isAuthenticated ? (
        <PrivyUserProfile showLogoutButton />
      ) : (
        <PrivyLoginButton>Sign In</PrivyLoginButton>
      )}
    </header>
  );
}
```
