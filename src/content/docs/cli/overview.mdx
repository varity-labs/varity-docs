VarietyKit is a Python-based CLI that helps you build, test, and deploy applications. If you have used tools like the Vercel CLI, Netlify CLI, or Firebase CLI, you will feel right at home.

## Commands at a Glance

| Command | Description |
|---------|-------------|
| `varitykit doctor` | Validate your development environment |
| `varitykit init` | Create a new project from a template |
| `varitykit app deploy` | Deploy your application |
| `varitykit localnet` | Manage local development network |
| `varitykit deploy` | Deploy backend services |

## Quick Start

```bash
# Install the CLI
pip install varitykit

# Check your environment
varitykit doctor

# Create a new project
varitykit init my-app

# Deploy (static sites)
varitykit app deploy

# Deploy (dynamic apps - coming soon)
varitykit app deploy --dynamic
```

## Global Options

All commands support these global options:

```bash
varitykit [OPTIONS] COMMAND [ARGS]...

Options:
  --version        Show the CLI version
  -v, --verbose    Enable verbose output
  --debug          Enable debug output
  --json           Output in JSON format
  --help           Show help message
```

## Requirements

- **Python**: 3.8 or higher
- **Docker**: 20.10 or higher (for localnet)
- **Node.js**: 18+ (for building JavaScript projects)

<Aside type="tip">
Run `varitykit doctor` to automatically check all requirements and get installation guidance for missing dependencies.
</Aside>

## Environment Variables

The CLI uses these environment variables for various features:

| Variable | Required For | Description |
|----------|--------------|-------------|
| `THIRDWEB_CLIENT_ID` | Deployment | Auto-provided by Varity (optional override) |
| `AKASH_WALLET_KEY` | Akash deployment | Key for Akash deployments |
| `WALLET_PRIVATE_KEY` | Backend deployment | Private key for signing operations |

## Hosting Options

### Static Sites

Deploy static websites and single-page applications:

```bash
varitykit app deploy
```

- **Best for**: Marketing sites, documentation, SPAs
- **Cost**: Free (sponsored by Varity)
- **Speed**: Global CDN distribution

### Dynamic Apps (Coming Soon)

Deploy full-stack applications with backend:

```bash
varitykit app deploy --dynamic
```

- **Best for**: APIs, databases, server-side rendering
- **Cost**: Pay for compute resources
- **Features**: Containers, scaling, persistence

<Aside type="note">
Dynamic hosting is under active development and will be available soon.
</Aside>
