## Community

Our Discord is the fastest way to get help from the team and other developers.

[Join Discord](https://discord.gg/Uhjx6yhJ)

Channels: **#help** for questions, **#general** for discussion, **#showcase** to share what you build.

## Report Issues

Found a bug or have a feature request? Open an issue on GitHub.

[Open an Issue](https://github.com/varity-os/varity-sdk/issues)

When reporting bugs, include:

- Steps to reproduce
- Expected vs actual behavior
- OS, Node version, and package versions
- Error messages or logs

## Security Issues

**Do not report security vulnerabilities publicly.**

Email **security@varity.so** instead. We respond within 24 hours.

## Contributing

Varity is open source. We welcome bug fixes, documentation improvements, and feature contributions.

[Contributing Guide](https://github.com/varity-os/varity-sdk/blob/main/CONTRIBUTING.md)
