All React components exported by `@varity-labs/ui-kit`.

## Providers

### PrivyStack

All-in-one provider that sets up authentication and account management. Wrap your app with this once.

```tsx
import { PrivyStack } from '@varity-labs/ui-kit';

<PrivyStack
  appId="your-privy-app-id"
  loginMethods={['email', 'google', 'wallet']}
  appearance={{
    theme: 'light',
    accentColor: '#2563EB',
    logo: '/logo.png',
  }}
>
  {children}
</PrivyStack>
```

**Props:**

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `appId` | `string` | Dev default | Privy App ID |
| `loginMethods` | `string[]` | `['email', 'google', 'wallet']` | Login options |
| `appearance` | `object` | Light theme | UI customization |
| `onAddressChange` | `function` | - | Account address callback |

## Authentication Components

### PrivyLoginButton

Pre-built login button that opens the authentication modal.

```tsx
import { PrivyLoginButton } from '@varity-labs/ui-kit';

// Basic usage
<PrivyLoginButton />

// With custom text
<PrivyLoginButton>Sign In to Continue</PrivyLoginButton>

// With callbacks
<PrivyLoginButton
  onSuccess={() => router.push('/dashboard')}
  onError={(error) => toast.error(error.message)}
  className="btn-primary"
/>
```

**Props:**

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | "Sign In" | Button text |
| `onSuccess` | `function` | - | Called on successful login |
| `onError` | `function` | - | Called on login error |
| `className` | `string` | - | CSS class name |

### PrivyUserProfile

Displays user information with optional logout button.

```tsx
import { PrivyUserProfile } from '@varity-labs/ui-kit';

<PrivyUserProfile showLogoutButton />

// With custom logout handler
<PrivyUserProfile
  showLogoutButton
  onLogout={() => {
    clearCache();
    router.push('/');
  }}
/>
```

**Props:**

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `showLogoutButton` | `boolean` | `false` | Show logout button |
| `onLogout` | `function` | - | Custom logout handler |
| `className` | `string` | - | CSS class name |

**Displays:**
- Account type (Email, Google, Twitter, etc.)
- Email or username
- Account address (shortened)
- Join date

### PrivyProtectedRoute

Shows content only to authenticated users.

```tsx
import { PrivyProtectedRoute } from '@varity-labs/ui-kit';

// Basic usage - shows login prompt when not authenticated
<PrivyProtectedRoute>
  <Dashboard />
</PrivyProtectedRoute>

// With custom fallback
<PrivyProtectedRoute
  fallback={<CustomLoginPage />}
  loadingComponent={<CustomSpinner />}
>
  <Dashboard />
</PrivyProtectedRoute>
```

**Props:**

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `ReactNode` | - | Protected content |
| `fallback` | `ReactNode` | Login prompt | Shown when not authenticated |
| `loadingComponent` | `ReactNode` | Spinner | Shown while loading |

### PrivyReadyGate

Prevents a blank screen during initialization (5-15 seconds).

```tsx
import { PrivyReadyGate } from '@varity-labs/ui-kit';

<PrivyReadyGate
  timeout={10000}
  initializingScreen={<SplashScreen />}
  timeoutScreen={<ConnectionError />}
>
  {children}
</PrivyReadyGate>
```

**Props:**

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `timeout` | `number` | `10000` | Timeout in milliseconds |
| `initializingScreen` | `ReactNode` | Loading spinner | Shown during init |
| `timeoutScreen` | `ReactNode` | Error message | Shown on timeout |

## Payment Components

:::caution[Beta Limitation]
Credit card payments are not available during beta. Apps can use the free tier. Paid tiers with credit card support will be available after production launch.
:::

### PaymentWidget

In-app payment widget for purchases and upgrades (available post-beta).

```tsx
import { PaymentWidget } from '@varity-labs/ui-kit';

<PaymentWidget
  appId={process.env.NEXT_PUBLIC_VARITY_APP_ID}
  amount={50}
  buttonText="Upgrade to Pro"
  onSuccess={(result) => {
    console.log('Payment complete:', result);
  }}
  onError={(error) => {
    console.error('Failed:', error);
  }}
/>
```

**Props:**

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `appId` | `string` | Required | Varity App ID |
| `amount` | `number` | `100` | Payment amount in USD |
| `buttonText` | `string` | "Purchase" | Button label |
| `theme` | `'light' \| 'dark'` | `'light'` | Color theme |
| `onSuccess` | `function` | - | Success callback |
| `onError` | `function` | - | Error callback |

### PaymentGate

Subscription and tier gate for paywalled content (available post-beta).

```tsx
import { PaymentGate } from '@varity-labs/ui-kit';

<PaymentGate
  appId={process.env.NEXT_PUBLIC_VARITY_APP_ID}
  tier="business"
  amount={99}
  onSuccess={() => console.log('Subscribed!')}
/>
```

**Props:**

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `appId` | `string` | Required | Varity App ID |
| `tier` | `string` | Required | Subscription tier name |
| `amount` | `number` | Required | Monthly price in USD |
| `onSuccess` | `function` | - | Success callback |
| `onError` | `function` | - | Error callback |

## Complete Example

```tsx
import {
  PrivyStack,
  PrivyLoginButton,
  PrivyUserProfile,
  PrivyProtectedRoute,
  usePrivy,
} from '@varity-labs/ui-kit';

function App() {
  return (
    <PrivyStack appId={process.env.NEXT_PUBLIC_PRIVY_APP_ID}>
      <Header />
      <main>
        <PrivyProtectedRoute>
          <Dashboard />
        </PrivyProtectedRoute>
      </main>
    </PrivyStack>
  );
}

function Header() {
  const { authenticated } = usePrivy();

  return (
    <header className="flex justify-between p-4">
      <h1>My App</h1>
      {authenticated ? <PrivyUserProfile showLogoutButton /> : <PrivyLoginButton />}
    </header>
  );
}

function Dashboard() {
  const { user } = usePrivy();

  return (
    <div>
      <h2>Welcome, {user?.email?.address}</h2>
    </div>
  );
}
```

:::note
Smart account components (`SmartWalletProvider`, `SmartWalletConnectButton`, `GaslessBadge`) will be available in a future release.
:::
